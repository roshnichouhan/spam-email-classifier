# 📧 Spam Email Classifier (No NLP)

## Features
- EDA + Feature Engineering
- Machine Learning Model
- FastAPI Backend
- Streamlit UI

## Run Project

### 1. Train model
python run.py

### 2. Run API

uvicorn backend.main:app --reload
 
### 3. Run UI
streamlit run app/streamlit_app.py
streamlit run app/streamlit_app.py
