# 📧 Spam Email Classifier 

 Project Overview

SpamShield AI is an end-to-end Machine Learning application that detects and classifies spam emails in real time using Natural Language Processing (NLP).

The platform combines a FastAPI backend with a modern Streamlit dashboard to provide an interactive and production-ready email security solution.

The system processes incoming email text, transforms it using TF-IDF vectorization, and predicts whether the message is Spam or Safe (Ham) using a trained Machine Learning model.

🎯 Business Problem

Email spam and phishing attacks remain among the most common cybersecurity threats.

Organizations receive thousands of emails daily, making manual filtering inefficient and risky.

SpamShield AI helps organizations:

✅ Detect spam emails automatically

✅ Reduce phishing risks

✅ Improve email security

✅ Increase employee productivity

✅ Enable real-time threat detection
